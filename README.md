## Welcome to the GitHub Page of project [OptiDrape](http://www.optidrape.de)

### Dependencies
1. Python 3.5+
2. PyQt 5
3. [openmesh 6.3](https://www.openmesh.org/)
4. NumPy
5. Matplotlib
6. h5py
7. pymongo
8. scipy
9. tqdm
10. qimage2ndarray
11. mpl_toolkits
12. tensorflow-gpu

### Start
To start the main script: 
'python3 gui/gui_main.py'
